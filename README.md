# パスワード生成ツール

暗号学的に安全なパスワードを生成し、その強度を評価するツールです。**CLI版**と**GUI版**の2つのインターフェースを提供します。

## 概要

このツールは、Pythonの`secrets`モジュールを使用して、予測不可能で安全なパスワードを生成します。生成されたパスワードは自動的に強度評価され、改善のためのフィードバックが提供されます。

- **CLI版** (`password_generator.py`): コマンドラインで素早くパスワードを生成
- **GUI版** (`password_generator_gui.py`): 視覚的なインターフェースで直感的に操作

## 特徴

### 共通機能
- **暗号学的に安全**: `secrets`モジュールによる予測不可能な乱数生成
- **柔軟な設定**: パスワードの長さと使用する文字種を自由に選択可能
- **強度評価機能**: 生成されたパスワードを自動評価し、スコアとフィードバックを提供

### CLI版の特徴
- **使いやすいCLI**: 直感的なコマンドラインインターフェース
- **複数生成対応**: 一度に複数のパスワードを生成可能
- **スクリプト連携**: 他のスクリプトやパイプラインと組み合わせて使用可能

### GUI版の特徴
- **視覚的なインターフェース**: Tkinterベースの使いやすいGUI
- **リアルタイム強度表示**: 色分けされたプログレスバーでパスワード強度を視覚的に表示
- **スライダー調整**: パスワード長をスライダーで直感的に調整
- **ワンクリックコピー**: クリップボードへ簡単にコピー可能
- **起動時自動生成**: アプリ起動時に自動的にパスワードを生成

## 機能一覧

### パスワード生成
- パスワード長の指定（デフォルト: 16文字）
- 文字種の選択
  - 大文字（A-Z）
  - 小文字（a-z）
  - 数字（0-9）
  - 記号（!@#$%など）
- 複数パスワードの一括生成

### 強度評価
パスワード強度は以下の3つの要素で100点満点で評価されます：

1. **長さ（最大40点）**
   - 16文字以上: 40点
   - 12-15文字: 30点
   - 8-11文字: 20点
   - 8文字未満: 10点

2. **文字種の多様性（最大40点）**
   - 4種類すべて使用: 40点
   - 3種類使用: 30点
   - 2種類使用: 15点
   - 1種類のみ: 5点

3. **エントロピー（最大20点）**
   - エントロピー = パスワード長 × log₂(文字セットサイズ)
   - 80以上: 20点
   - 60-79: 15点
   - 40-59: 10点
   - 40未満: 5点

### 評価レベル
- **非常に強い**: 80点以上
- **強い**: 60-79点
- **中程度**: 40-59点
- **弱い**: 40点未満

## インストール方法

このツールはPython 3.7以上が必要です。

### 基本インストール（CLI版のみ）

CLI版は標準ライブラリのみを使用しているため、追加のパッケージインストールは不要です。

```bash
# リポジトリをクローン
git clone https://github.com/nova17x/password-generator.git
cd password-generator

# 実行権限を付与
chmod +x password_generator.py
```

### GUI版を使用する場合

GUI版を使用するには、追加でTkinterのインストールが必要です（Linuxの場合）。

**Ubuntu/Debian系:**
```bash
sudo apt-get update
sudo apt-get install python3-tk
```

**Fedora/RHEL系:**
```bash
sudo dnf install python3-tkinter
```

**macOS:**
通常、Pythonに標準で含まれているため、追加インストール不要です。

**Windows:**
通常、Pythonに標準で含まれているため、追加インストール不要です。

## 使い方

### GUI版の使い方

GUI版は視覚的なインターフェースで直感的に操作できます。

#### 起動方法

```bash
python3 password_generator_gui.py
```

#### 使用手順

1. **パスワードの長さを調整**: スライダーを動かして4〜32文字の範囲で設定
2. **文字種を選択**: 大文字、小文字、数字、記号のチェックボックスをON/OFF
3. **「パスワードを生成」ボタンをクリック**: 新しいパスワードが生成されます
4. **パスワードを確認**: 表示エリアに生成されたパスワードが表示されます
5. **クリップボードにコピー**: 「クリップボードにコピー」ボタンをクリック
6. **強度を確認**: 色付きプログレスバーとスコアで強度を視覚的に確認

#### GUI版の画面構成

- **パスワード長さスライダー**: 4〜32文字の範囲で調整可能
- **文字種選択**: 大文字/小文字/数字/記号のチェックボックス
- **パスワード表示エリア**: 生成されたパスワードを表示
- **強度インジケーター**:
  - 緑: 非常に強い (80-100点)
  - 黄緑: 強い (60-79点)
  - オレンジ: 中程度 (40-59点)
  - 赤: 弱い (0-39点)

### CLI版の使い方

CLI版はコマンドラインから素早くパスワードを生成できます。

#### 基本的な使い方

```bash
# デフォルト設定で1つ生成（16文字、全文字種使用）
python3 password_generator.py
```

出力例:
```
パスワード 1: 4OA!2G^xBAU?-t1x
強度: 非常に強い (スコア: 100/100)
評価: 良好なパスワードです
```

#### オプション付きの使用例

```bash
# 20文字のパスワードを3個生成
python3 password_generator.py -l 20 -n 3

# 記号を含まない12文字のパスワード
python3 password_generator.py -l 12 --no-symbols

# 大文字と数字のみの10文字パスワード
python3 password_generator.py -l 10 --no-lower --no-symbols

# ヘルプを表示
python3 password_generator.py --help
```

#### コマンドラインオプション

| オプション | 短縮形 | 説明 | デフォルト |
|-----------|--------|------|-----------|
| `--length` | `-l` | パスワードの長さ | 16 |
| `--count` | `-n` | 生成する個数 | 1 |
| `--upper` / `--no-upper` | - | 大文字を含む/含まない | True |
| `--lower` / `--no-lower` | - | 小文字を含む/含まない | True |
| `--digits` / `--no-digits` | - | 数字を含む/含まない | True |
| `--symbols` / `--no-symbols` | - | 記号を含む/含まない | True |
| `--help` | `-h` | ヘルプを表示 | - |

## CLI版とGUI版の使い分け

どちらを使用するか迷った場合の参考にしてください。

### GUI版がおすすめの場合
- **初めて使う方**: 視覚的で直感的なインターフェース
- **1つのパスワードを生成したい**: ワンクリックで生成・コピー
- **強度を視覚的に確認したい**: 色付きプログレスバーで一目で判断
- **設定を調整しながら試したい**: スライダーで簡単に長さを変更

### CLI版がおすすめの場合
- **複数のパスワードを一度に生成したい**: `-n`オプションで一括生成
- **スクリプトと組み合わせたい**: 他のコマンドとパイプラインで連携
- **素早く生成したい**: コマンド一発で即座に生成
- **サーバー環境で使いたい**: GUIが不要な環境での使用

## セキュリティについて

このツールは以下のセキュリティベストプラクティスに従っています：

- **`secrets`モジュールの使用**: Pythonの`random`モジュールではなく、暗号学的に安全な`secrets`モジュールを使用
- **予測不可能性**: 生成されるパスワードは統計的に予測不可能
- **強度評価**: 生成されたパスワードの強度を客観的に評価し、改善点を提示

## 技術仕様

- **言語**: Python 3.7+
- **依存関係**:
  - **CLI版**: 標準ライブラリのみ
    - `secrets`: 暗号学的に安全な乱数生成
    - `string`: 文字セット定義
    - `argparse`: コマンドライン引数解析
    - `math`: エントロピー計算
  - **GUI版**: CLI版 + Tkinter
    - `tkinter`: GUIフレームワーク
    - `password_generator`: CLI版の関数を再利用
